# Backend

## Installation

Run

```
npm install
```

in `backend` directory to build the server

## Development

Run

```
npm start
```

or

```
npm run start:dev
```

in `backend` directory to start the server on port 4000

## Build

Run

```
npm run build
```

in `backend` directory to build the server

---

# Frontend

## Installation

Run

```
npm install
```

in `frontend` directory to build the client

## Development

Run

```
npm run dev
```

in `frontend` directory to start the client on port 3000

## Build

Run

```
npm run build
```

in `frontend` directory to build the client
